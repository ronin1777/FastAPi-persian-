# مقدمه‌ای بر انواع داده‌های پایتون

زبان پایتون از **"type hints"** اختیاری پشتیبانی می کند (که همچنین به عنوان **"type annotations"** نیز شناخته می شود).<br>
این **"type hints"** یا annotations سینتکس خاصی هستند که امکان اعلام نوع یک متغیر را فراهم می کنند.<br>
با اعلام نوع برای متغیرهای خود، ویرایشگرها و ابزارها می توانند پشتیبانی بهتری را به شما ارائه دهند.<br>
این فقط یک آموزش سریع یا مروری در مورد "type hints" پایتون است. این آموزش تنها موارد ضروری برای استفاده از آنها با **"FastAPI"** را پوشش می دهد... که در واقع بسیار کم است.
FastAPI به طور کامل بر اساس این "type hints" استوار است، آنها مزایای زیادی را به آن می دهند.
اما حتی اگر هرگز از **FastAPI** استفاده نکنید، دانستن اندکی در مورد آنها برای شما مفید خواهد بود.
> [!NOTE]
> اگر یک متخصص پایتون هستید و قبلاً همه چیز در مورد "type hints" را می دانید، به فصل بعدی بروید.
